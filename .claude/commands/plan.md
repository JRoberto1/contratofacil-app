# /plan — Decomposição em Tarefas

Decomponha em tarefas com critérios verificáveis.

## Instrução
Leia a spec em docs/specs/ se existir. Crie tabela com: tarefa, arquivo(s), critério verificável, dependência.
Aguarde aprovação antes de executar.

## Anti-Rationalization
❌ "As tarefas são óbvias" → Se é óbvio, leva 2min escrever. Escreva.
❌ "Posso fazer tudo de uma vez" → Tarefas grandes = contexto pesado = erros
