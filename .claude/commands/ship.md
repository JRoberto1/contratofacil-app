# /ship — Checklist de Produção

Não faça deploy sem este checklist.

## Checklist
- [ ] /review executado e aprovado
- [ ] Todos os testes passando
- [ ] Sem console.log em produção
- [ ] .env não commitado
- [ ] docs/architecture.md atualizado se houve decisão nova
- [ ] Build passa localmente
- [ ] Plano de rollback definido

## Anti-Rationalization
❌ "É urgente, vou pular alguns" → Urgência não cancela segurança
❌ "É só um hotfix" → Hotfixes sem checklist geram hotfixes de hotfix
